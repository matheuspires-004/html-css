Principais Formatações

Negrito / Destaque

Nessa frase, temos um termo em negrito usando a tag <b>(não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong>(semântica).

Itálico / Enfâse

Nesta frase, temos um termo em itálico usando a tag <i>(não semântica).

Nesta frase, temos um termo em enfâse usando a tag <em>(semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo temos outro texto marcado no final.

Texto grande

Estamos criando um texto grande, usando a tag <big>(obisoleto), e um texto pequeno, usando a tag <small> nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído, usando a tag <del>, para indicar que ele deve ser lido, mas não considerado.

Texto Inserido / Sublinhado

Podemos marcar um texto como inserido, usando a tag <ins>, para dar uma enfâse e indicar que ele foi adicionado depois(semântica).

Existe também o texto sublinhado, usando a tag <u>, (não semântica).

Texto Sobrescrito

Para inserir coisas do tipo x20+3, usando a tag <sup>(caractere em cima).

Texto Subscrito

Para inserir coisas do tipo H2O, usando a tag <sub>(caractere embaixo).
